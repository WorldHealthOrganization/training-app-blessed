Features

Tutorials for custom data processes

Build and provide users with custom tutorials to guide them through specific data collection process

Off-the-self tutorials for DHIS2 core apps

Use Training App’s off-the-self tutorials to help users learn how to use DHIS2 core apps

Tutorial translation

Translate tutorials to the languages of your DHIS2 users

Custom landing pages

Define how users will land on your tutorials by creating custom landing pages

Tutorials for WHO standard DHIS2 metadata package

Download tutorials in the offical GitHub repository
